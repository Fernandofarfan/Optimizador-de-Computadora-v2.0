## Cómo Usar las Opciones Funcionales

### Opción [1] - Analizar Sistema
```
Muestra:
- Información del SO y procesador
- Estado de RAM (total y usado)
- Estado de discos (espacio libre)
- Cantidad de servicios activos
```

### Opción [2] - Limpieza Rápida
```
Ejecuta:
- Limpieza básica y segura
- Sin riesgos de perder datos
- Ideal para mantenimiento regular
```

### Opción [3] - Limpieza Profunda
```
Opciones:
1. Limpiar archivos temporales
2. Vaciar papelera
3. Limpiar cache (pendiente)
4. Ejecutar todas las limpiezas
```

## Solución de Problemas

Si una opción no responde:
1. Asegúrate de tener permisos de Administrador
2. Verifica que se esté ejecutando desde PowerShell elevado
3. Intenta de nuevo con la opción [0] para volver al menú
