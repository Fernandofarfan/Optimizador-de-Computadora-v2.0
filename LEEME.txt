DESCRIPCION:
------------
Optimizador de PC v2.0 es una suite completa de herramientas para mantener,
optimizar y monitorear tu sistema Windows. Incluye 42 funciones 100% operativas
para mejorar el rendimiento y estabilidad de tu computadora.

VERSION 2.0 - MEJORAS:
- 20 funciones corregidas y optimizadas
- 6 modulos completamente recreados
- 4,000+ lineas de codigo obsoleto eliminadas
- 0 errores en todo el proyecto
- GitHub Pages profesional actualizado


REQUISITOS DEL SISTEMA:
-----------------------
- Windows 10 o Windows 11
- PowerShell 4.0 o superior
- Permisos de Administrador (para la mayoria de funciones)
- Minimo 4 GB de RAM
- 100 MB de espacio en disco


INSTALACION:
------------
1. Extrae todos los archivos en una carpeta de tu eleccion
2. Ejecuta "EJECUTAR-OPTIMIZADOR.bat" para iniciar el programa principal
3. Acepta los permisos de administrador cuando se soliciten


ARCHIVOS PRINCIPALES:
---------------------
- EJECUTAR-OPTIMIZADOR.bat : Inicia el optimizador principal
- PROBAR-FUNCIONES.bat     : Modo de prueba individual de funciones
- Optimizador.ps1          : Programa principal con menu interactivo
- config.json              : Archivo de configuracion

MODO DE USO:
------------
1. Ejecuta EJECUTAR-OPTIMIZADOR.bat
2. Selecciona una opcion del menu escribiendo su numero
3. Presiona ENTER para ejecutar
4. Sigue las instrucciones en pantalla
5. Para salir escribe 0 y presiona ENTER


MODO PRUEBA:
------------
Si deseas probar funciones individuales sin pasar por el menu principal:
1. Ejecuta PROBAR-FUNCIONES.bat
2. Selecciona el numero de la funcion que deseas probar
3. La funcion se ejecutara directamente

SOLUCION DE PROBLEMAS:
----------------------
Problema: No se ejecuta el programa
Solucion: Verifica que tienes PowerShell instalado y permisos de administrador

Problema: Error de ejecucion de scripts
Solucion: Ejecuta en PowerShell (como admin):
          Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Problema: Funciones no responden
Solucion: Cierra PowerShell y vuelve a ejecutar EJECUTAR-OPTIMIZADOR.bat
